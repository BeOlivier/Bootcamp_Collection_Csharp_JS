## CSS lab

A common task as a frontend developer is to style the HTML layer after some wireframe, or even a complete design made by graphical designers.
When doing that, you must try to be as close to the original idea as possible.

- Stick to the same layout and colors.
- If you know what font is used, then use that. If not, try to find something close enough.

### CSS fundamentals

In this lab, you will practice the CSS fundamentals.
That means that newer additions to CSS like `Flexbox` or `Grid` __is not allowed to use in this lab!__

## 1) News Assembly

The _News Assembly_ is a news aggregator service that collects all the important news for its users.
The company has a serious profile, yet they want a fresh feel that attracts users who typically not visit the classic news sites.

The company color profile is matte pink/gray `#ccaabb` on a white `#fff` background.
The logotype is using the google font `Vidaloka` (serif).
However, they want the page to have a more relaxed feel, so they want you to use another font for the content. (See the example)

![News Assembly](assets/newsassembly.png?raw=true "News Assembly")

## 2) The Jungle

The _The Jungle_ is a tiki metal bar located in a midsize city.
They are an established business that has gone through a small rebranding and now they want to attract both the old metal audience as well as a newer audience.
The restaurant has a simple bar menu, which is ok, but the main reason it is a popular bar is because of the live music.
But the tiki bar is new and the new owner has high hopes for attracting an even wider audience.

![The Jungle](assets/tikimetalbar.png?raw=true "The Jungle")

## 3) My Awesome Company

_My Awesome Company_ is a low budget advice consultancy.
They want a simple and functional website that works like a live dashboard with articles.
The user should interact with the dashboard by scrolling.
They also provide a few links to some of their partners.
While scrolling, the header and links must remain in a fixed position on the page.
